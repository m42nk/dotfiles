## Dotfiles with Ansible

## Usage
**Build Image**
```bash
make build
```

**Run Container**
```bash
make run
```

**Build and Run Container**
```bash
make build-run
# Or
make br
```

**Prepare Ansible**
```bash
make prepare
```

**Run Ansible**
```bash
make play
```
